GP-LVM Software

This page will describe examples of how to use the Gaussian process latent variable model Software (GPLVM) available for download here. Alternative GP-LVM implementations from this site: The GP-LVM C++ software is available from here. A new MATLAB version of the toolbox is now being worked on. It doesn't rely on the IVM toolbox, and currently will only do Gaussian noise models. The sparse algorithm is a development of Snelson and Ghahramani's pseudo-inputs. This new version is available here.

Release Information
